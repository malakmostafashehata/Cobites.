# 🍽️ CO-BITES

## 📌 Overview
CO-BITES is a smart food logistics and redistribution platform designed to reduce food waste by connecting food donors with charities and delivery personnel in real time.

The system ensures that surplus food from restaurants, individuals, and grocery stores is efficiently collected and delivered to people in need through a structured digital workflow.

---

## 🎯 Project Mission
To minimize food waste and hunger by creating a fast, transparent, and efficient system that connects food sources with charities and volunteers.

---

## ❗ Problem Statement
Despite widespread hunger, large amounts of edible food are wasted daily due to:

- Lack of a centralized communication system between donors and charities  
- Slow delivery coordination leading to food expiration  
- Poor visibility of available food resources  
- Environmental pollution caused by food waste  
- No tracking system for donor contributions  

---

## ⚙️ System Features

### 👤 User Management
- Secure registration and login system
- Role-based access (Admin, Charity, Volunteer, Delivery Person)
- Profile management

---

### 🍱 Donation Management
- Add food donations with:
  - Food type
  - Quantity
  - Expiry date
  - Pickup location
- Track all donations in the system

---

### 🏥 Charity Requests
- Browse available donations
- Request needed food items
- Automatic matching system between donations and requests

---

### 🚚 Delivery & Tracking
- Assign delivery tasks to volunteers
- Track delivery status:
  - Pending
  - In Transit
  - Delivered

---

### 📩 Contact Us System
- Users can send messages to admin
- Admin handles inquiries and support requests

---

### 🛠️ Admin Panel
- Manage users, donations, and deliveries
- Verify charities using official documents
- Monitor system activity and performance

---

### 📂 Data Storage
- File handling (I/O)
- Persistent storage of users, donations, and transactions

---

## 🌐 Technologies Used
- PHP (Backend)
- JavaScript (Frontend logic)
- HTML & CSS (UI design)
- File Handling (Data storage)
- Web-based application

---

## 🔐 Charity Verification System
- Charities must upload official documents during registration
- Admin verifies legitimacy before approval
- Only verified charities can access the system

---

## 📊 System Benefits
- Reduces food waste
- Supports people in need
- Improves donation transparency
- Enhances delivery efficiency
- Encourages community participation
